# Medical Clinic Management System 🏥

## Overview
A comprehensive relational database management system designed to streamline clinic operations, including patient registration, appointment scheduling, and medical record management. 

This project was developed as a core component of my Computer Science curriculum and received a **Final Grade of 100**.

## Key Features
* **Relational Database Design:** Fully normalized tables ensuring data integrity and reduced redundancy.
* **Complex Queries:** Advanced SQL-based queries for patient tracking, visit summaries, and clinic statistics.
* **User Interface:** Custom-built Forms for intuitive data entry and navigation.
* **Automated Reporting:** Professional reports generated for daily clinic schedules and patient histories.

## Technical Stack
* **Database Engine:** Microsoft Access
* **Query Language:** SQL
* **Tools:** MS Office Suite
